<model-viewer> - Accessibility Tests

There are several different tests in order to investigate different aspects of accessibility and <model-viewer>. This page enumerates the test cases, and provides links to the individual scenarios.

Note that all these examples include alt.

  • accessibility/poster.html: A single sample exercising poster.
  • accessibility/reveal.html: A single sample exercising poster, preload, and reveal-when-loaded.
  • accessibility/controls.html: A single sample exercising controls.
  • accessibility/auto-rotate.html: A single sample exercising auto-rotate.
  • accessibility/multiple.html: Multiple models on the same page with a variety of attributes.
  • augmented-reality.html: Test using configurations that enable viewing the model in augmented reality.
